<rect>

Description

<rect x="[x]" y="[y]" width="[width]" height="[height]" rx="[rx]" ry="[ry]" >
사각형을 생성함
속성
X : 사각형의 좌측 상단 X위치
Y : 사각형의 좌측 상단 Y위치
width : 사각형의 폭
width : 사각형의 높이
rx : 가로 방향 외각선 곡률
ry : 세로 방향 외각선 곡률

code

<svg> <rect width="100" height="100" /> <rect x="100" y="100" width="100" height="100" rx="50" ry="20" /> </svg>
