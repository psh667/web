[transform]

Description

[transform];
SVG 형태를 변경함
속성 값
translate([tx],[ty]) : 지정된 요소 좌표계를 수평 이동시킴
scale([sx,sy]) : 지정된 요소 좌표계의 스케일을 변경함
skewX([d]) : 지정된 요소 좌표계를 X축에 대해 기울임
skewY([d]) : 지정된 요소 좌표계를 Y축에 대해 기울임
rotate([d]) : 지정된 요소의 좌표계를 회전함
matrix([변환매트릭스]) : 변환매트릭스를 사용하여 좌표계특성을 변환함
