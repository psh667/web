<text x="[x]" y="[y]">[문자열]</text>
문자의 작성
속성
x : 문자열 위치의 X좌표
y : 문자열 위치의 Y좌표
<tspan dx["dx"] dy="[dy]">[문자열]</tspan>
문자열 그룹의 작성
속성
x : 문자열 위치의 X좌표
y : 문자열 위치의 Y좌표
dx : 원래 문자열 위치에서 변경할 X좌표 거리
dy : 원래 문자열 위치에서 변경할 Y좌표 거리
