<image>

Description

<image x="[x]" y="[y]" width="[width]" height="[height]" xlink:href="이미지 경로" />
SVG내로 이미지를 삽입함
속성
x : 이미지가 삽입되는 좌측상단의 X 좌표
y : 이미지가 삽입되는 좌측상단의 Y 좌표
width : 이미지가 삽입되는 폭
height : 이미지가 삽입되는 높이
xlink:href : 삽입되는 이미지의 경로

code

<!--SVG부분--> <svg> <image x="100" y="5" width="204" height="189" xlink:href="img/coforwardQr2.png" /> </svg>
