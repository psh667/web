min-width & min-height

Description

min-width:[length]; / min-height:[length];
요소의 최소 폭과 최소 높이를 지정한다.
Contents의 높이값이 지정값 이상이면 해당요소의 속성값이 확장됨

Code

<style type="text/css"> #exampleSub div{ min-height:100px; width:50%; } #box01{ border:#333 double 3px; color:#F00; } #box02{ border:#333 double 3px; background:#FF3; color:#999; } </style>
