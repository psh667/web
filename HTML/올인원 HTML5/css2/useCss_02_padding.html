Padding

Description

padding:[length];
Content와 Border사이의 여백
padding-top / padding-right / padding-bottom/ padding-left으로 개별 지정 가능
border-width / border-style / border-color 로 개별적 속성지정 가능
background 속성이 적용됨

Code

<style type="text/css"> #inBox01{ background:#0F9; margin:30px; padding:30px; } #inBox01 p{ background:#CCC; } </style>
