white-space

Description

white-space:[value];
공백처리에 대한 설정을 지정한다.

Code

<style type="text/css"> #par01{ background:#0FC; white-space:normal; } #par02{ background:#CF6; white-space:pre; } #par03{ background:#F9F; white-space:nowrap; } </style>

Example
