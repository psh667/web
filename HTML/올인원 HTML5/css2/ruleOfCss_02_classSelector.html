Class Selector

Description

class선택자
한번에 여러개의 요소에 같은 속성을 지정할 수 있다.

Code

<style type="text/css"> span{ color:#f00; } .doubleBorder{ border:#000 double 3px; } </style>

Example

[h3] element class="doubleBorder"

[p] elemet

[p] elemet[p-span] element class="doubleBorder"

[p] element class="doubleBorder"[p-span] element

[div] element class="doubleBorder"

[div-p] element

[div-p] element [div-p-a element] class="doubleBorder"
