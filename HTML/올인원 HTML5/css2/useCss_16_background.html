background

Description

background : [value] [value] [value]..;
배경에 관련된 속성을 일괄 지정한다.
예제에 사용된 image : check.png

Code

<style type="text/css"> #box01{ background:url(img/check.png); } #box02{ background:url(img/check.png) no-repeat 50% 50% #0FC; } #box03{ background:url(img/check.png) repeat-x 20px #FCC; } #box04{ background:url(img/check.png) repeat-y 40px #FF0; } </style>

Example

[div] element "box01"
[div] element "box02"
[div] element "box03"
[div] element "box04"
