border

Description

border:[length];
Box Model에 외각선을 그림
border-top / border-right / border-bottom / border-left으로 개별 지정 가능
border-width / border-style / border-color 로 개별적 속성지정 가능
속성별 개별지정도 가능함 : border-left-width:10px;

Code

<style type="text/css"> #outBox{ width:400px; border:#f00 solid 5px; background:#CCC; } #inBox01{ background:#0F9; margin:30px; border-top:#000 solid 10px; border-left:#F00 dashed 10px; border-right:#00F double 10px; border-bottom:#000 dotted 10px; } </style>
