CSS3 : background-clip

Description

background-clip : [keyword];
백 그라운 이미지 표시 영역을 지정함.
속성 값
keyword :
  • padding-box : padding영역부터 까지 배경 이미지를 표시함 (기본값)
  • border-box : border영역까지 배경 이미지를 표시함

Code

<style type="text/css"> #border-box{ background:url(img/photo3.gif); background-clip:border-box; -webkit-background-clip:border-box; -moz-background-clip:border-box; } #padding-box{ background:url(img/photo3.gif); background-clip:padding-box; -webkit-background-clip:padding-box; -moz-background-clip:padding-box; } </style>
