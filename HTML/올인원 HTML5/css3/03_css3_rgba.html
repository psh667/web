CSS3 : rgba & hsla

Description

rgba([r],[g],[b],[a]);
투명정도 지정할 수 있는 색상 입력값
입력 값
r : 빨간색의 수치 0~255
g : 녹색의 수치 0~255
b : 파란색의 수치 0~255
a : 투명도 0~1
hsla([h],[s],[l],[a]);
투명정도 지정할 수 있는 색상 입력값
입력 값
h : 색상 값 0~360
s : 채도 0~100%
l : 명도 0~255
a : 투명도 0~1

Code

<style type="text/css"> #rgbaBox .coFrowardText{ position:absolute; top:30px; left:30px; background:rgba(108,173,223,0.5); font-size:50px; } #rgbaBox .html5Css3Text{ width:250px; position:absolute; top:90px; left:70px; background:url(img/checkBox4.png) repeat; color:hsla(0,100%,60%,0.5); font-size:40px; } </style>
