CSS3 : resize

Description

resize
요소의 크기를 사용자가 변경할수 있도록함
속성 값
horizontal : 가로방향의 크기를 변경을 할수 있음
horizontal : 세로방향의 크기를 변경을 할수 있음
both : 양 방향의 크기를 변경을 할수 있음

Code

<style type="text/css"> #exampleSub div{ margin:0 auto; margin-bottom:15px; border:#999 solid 2px; width:70%; padding:20px; } #resize_Box{ resize:both; overflow:auto; } </style>
