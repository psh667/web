CSS3 : box-size

Description

box-sizing
요소박스 크기를 지정하는 방법
속성 값
content-box : width와 height는 content만의 크기로 적용됨 (CSS2표준방법)
border-box : width와 height는 padding과 border를 포함한 크기로 적용됨 (기존 IE 모드)

Code

<style type="text/css"> #contentBox{ width:60%; padding:10px; border:#096 solid 20px; box-sizing:content-box; -webkit-box-sizing:content-box; -moz-box-sizing:content-box; } #borderBox{ width:60%; padding:20px; border:#096 solid 20px; box-sizing:border-box; -webkit-box-sizing:border-box; -moz-box-sizing:border-box; } #floatBox{ width:60%; padding:0 20px; height:40px; border:#096 solid 20px; } .bBoxfloat{ width:50%; padding:10px; border:#999 solid 10px; box-sizing:border-box; -webkit-box-sizing:border-box; -moz-box-sizing:border-box; float:left; } </style>
