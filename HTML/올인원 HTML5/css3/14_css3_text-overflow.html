CSS3 : text-overflow

Description

text-overflow:[keyword];
오버플로어 되는 텍스트의 표시를 지정
속성 값
clip : 지정된 영역의 크기로 잘라낸다.
ellipsis : 지정된 영역의 크기에 맞게 말줄임표를 표시한다.

Code

<style type="text/css"> #textBox p{ margin:0 auto; margin-bottom:10px; border:1px solid #ccc; padding:10px; width:90%; white-space:nowrap; overflow:hidden; } #textBox .clipText{ text-overflow:clip; -o-text-overflow:clip; } #textBox .ellipsisText{ text-overflow:ellipsis; -o-text-overflow:ellipsis; } </style>

Example

clip

영역을 벗어나는 텍스트를 오버플로된 텍스트라 하는데 이렇게 오버플로된 텍스트를 생략부호(...)로 처리하는 속성입니다.

ellipsis

영역을 벗어나는 텍스트를 오버플로된 텍스트라 하는데 이렇게 오버플로된 텍스트를 생략부호(...)로 처리하는 속성입니다.
