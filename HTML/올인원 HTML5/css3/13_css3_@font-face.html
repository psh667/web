CSS3 : @font-face

Description

@font-face{}
서버측 폰트를 사용할 수 있는 font-family를 생성함

Code

<style type="text/css"> @font-face{ font-family:nanumFont; src:url('img/nanum.eot'); src:local('nanumgothicExtraBold'), url('img/nanum.ttf') format('truetype'); } #textBox .ex01{ font-family:"돋움",dotum; border:1px solid #ccc; padding:10px; } #textBox .ex02{ font-family:"굴림",gulim; border:1px solid #ccc; padding:10px; } #textBox .ex03{ font-family:nanumFont,gulim; border:1px solid #ccc; padding:10px; } </style>

Example

돋움, dotum

굴림, gulim

나눔고딕, nanum gothic
