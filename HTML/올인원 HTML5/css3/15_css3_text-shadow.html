CSS3 : text-shadow

Description

text-shadow:[dx] [dy] [bulr] [color];
텍스트에 그림자를 지정함
속성 값
dx : 본체와 그림자의 가로방향 거리
dy : 본체와 그림자의 세로방향 거리
bulr : 그림자의 흐려짐 정도
color : 그림자의 색상

Code

<style type="text/css"> #html5Css3Text{ background:#afbc22; text-shadow:3px 3px 5px #000; } #coforwardText{ background:#aaa; text-shadow: 0 0 4px #ccc, 0 -3px 4px #ff3, 1px -6px 6px #fd3, -1px -9px 11px #f80, 1px -12px 18px #f20; } </style>
