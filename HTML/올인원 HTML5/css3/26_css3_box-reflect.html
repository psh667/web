CSS3 : box-reflect

Description

box-reflect:[반사 방향],[offset 거리],[마스크]
개체에 대한 반사 효과를 지정함
속성 값
반사방향 : 반사되는 방향을 지정함 above/below/left/right
offset거리 : 반사체가 원본과 떨어지는 거리를 지정
마스크 : 반사체 표시 부분에 마스크를 지정 할 수 있음

Code

<style type="text/css"> #ref_above{ -webkit-box-reflect:above 1px; } #ref_below{ -webkit-box-reflect:below 1px; } #ref_left{ -webkit-box-reflect:left 1px; } #ref_right{ -webkit-box-reflect:right 1px; } #sample img{ margin-bottom:50px; -webkit-box-reflect:below 1px -webkit-gradient(linear,left top,left bottom, from(transparent),color-stop(0.7,transparent),to(white)); } </style>
