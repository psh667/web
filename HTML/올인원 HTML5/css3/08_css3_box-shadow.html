CSS3 : box-shadow

Description

box-shadow:[dx] [dy] [bulr] [spread] [set];
지정된 요소의 그림자를 설정함
속성 값
dx : 그림자의 가로방향 위치
dy : 그림자의 세로방향 위치
bulr : 흐려짐 정도
spread : 번짐 정도
set : 그림자의 형태
  • outset : 요소 바깥쪽으로 그림자가 떨어짐 (기본값)
  • inset : 요소 안쪽으로 그림자가 떨어짐

Code

<style type="text/css"> #Box01{ box-shadow:10px 10px 5px 5px gray; -webkit-box-shadow:10px 10px 5px 5px gray; -moz-box-shadow:10px 10px 5px 5px gray; } #Box02 img{ box-shadow:10px 10px 5px 5px gray inset; -webkit-box-shadow:10px 10px 5px 5px gray inset; -moz-box-shadow:10px 10px 5px 5px gray inset; } </style>
