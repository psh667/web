CSS3 : outline-offset

Description

outline
요소의 아웃라인을 그림 border와는 별개의 속성이며 border를 설정한는 것과 같이 아래의 속성을 단축하여 사용할 수 있음
outline-style : 아웃라인의 형식
outline-color : 아웃라인의 색상
outline-width : 아웃라인의 두께
outline-offset:[d];
d : 아웃라인을 옵셋시키는 거리로 음수값도 가능함

Code

<style type="text/css"> #box_coForward{ background:#d9da55; border:#999 solid 5px; outline-style:dashed; outline-width:5px; outline-color:#676300; outline-offset:-20px; } #box_html5Css3{ background:#afbc22; border:#999 solid 5px; outline:#676300 double 5px; outline-offset:20px; } </style>
