CSS3 : box-flex

Description

box-flex
Flexible Box안에서 확장여부를 결정함
속성 값
0 : 박스를 확장하지 않음
1 : 박스를 확장함

Code

<style type="text/css"> #horizontalBox div{ width:25%; } .inBox01{ height:auto; background:#F96; } .inBox02{ height:auto; background:#0C3; } .inBox03{ background:#39F; box-flex:1; -webkit-box-flex:1; -moz-box-flex:1; } </style>

Example

box-orient:vertical

height:auto;
height:auto;
height:auto; box-flex:1;

box-orient:horizontal

width:25%;
width:25%;
width:25%;
box-flex:1;
