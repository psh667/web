Description

<sorce>
<video> 의 자원경로를 지정 함 여러개를 지정할수 있으나 브라우저에서 실행할 수 있는 하나만 재생됨
적용되는 속성
  • src : 비디오 자원의 경로를 지정함 <source> 를 사용할 경우 <video>에 [src]속성은 작성하지 않음
  • type : 비디오 코덱정보의 표시
  • media : 미디어 쿼리를 이용하여 전달할 소스를 달리 적용할수 있음
