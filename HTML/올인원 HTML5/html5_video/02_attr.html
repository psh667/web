<video> 요소 속성

Description

<video> 에 적용되는 속성들
  • src : 비디오 자원의 경로를 지정함
  • width : 비디오 요소의 넓이
  • height : 비디오오소의 높이
  • poster : 포스터 이미지의 지정
  • controls : 컨트롤페널의 표시
  • autoplay : 자동재생
  • loop : 반복재생
  • type : 비디오 코덱정보의 표시

code

<video src="source/area88.ogv" controls="controls" poster="img/poster.png" />
